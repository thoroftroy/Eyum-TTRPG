6.4.4 Vindroth Runes

6.4.4 Vindroth Runes

Dec 15, 20256 min read

Vindroth runes are written as sentence structures (often called paragraphs), allowing rune magic to function like a programmed trigger system: when a condition is met, an effect occurs.

Vindroth runes are not arranged in circles. Instead, they are inscribed as ordered “words” that form a readable sequence (ex: If … Than … Else …).

Inscription Rules

  • Each Vindroth rune occupies 1 square inch of space.
  • All runes in a paragraph must be inscribed into one continuous piece of material.
  • If the material has a complete break that separates words, the paragraph loses the separated words. This may weaken, change, or fully disable the effect.

Vindroth Structure Limits

A Vindroth paragraph is limited by your Vindroth School Level.

  • You may include 1 “If … Than” block per Vindroth School Level.
  • Each Vindroth School Level also increases how many logic-connectors you may include (Else / And / Or / Not).
  • At level 19, the maximum increases from 6 to 7 and at 20 it raises again to 8. From this point every 2 levels of effective Runemaster Level raises the maximum by 1.

Rune Knowledge

  • If and Than runes are always known and do not count against total runes known.

Using Xeran as Effect Payloads

Vindroth can call Xeran runes as effect payloads (usually to define damage type / element / output). Example structure:

  • If object (human) entered size (5) shape (sphere) Than Power (Xeran Fire) To increase output, you may repeat effect runes:
  • … Than Power (Xeran Fire) And Power (Xeran Fire) With each And Power adding another damage as well as consuming 1 Structure Limit

Vindroth Conditionals

RuneEffect
IfBegins a conditional check.
ThanExecutes the next statement(s) if the condition is true.
ElseExecutes the next statement(s) if the previous condition was false (may be followed by If).
AndCombines conditions/effects; both must apply (when used with If) or both occur (when used with effects).
OrCombines conditions; either condition being true is sufficient.
NotNegates a condition (true only if the condition is not met).
atSpecifies the point of origin / location where the effect occurs.
whileKeeps the effect active as long as the condition remains true.
sizeDefines area size; maximum area is 5 × Vindroth School Level.
shapeDefines area shape (sphere, cube, spear (directional attack)).
objectDefines what is being checked or targeted (leave blank to mean “any”).
withinTrue if the object is within the specified range/area.
enteredTrue if the object enters the specified range/area.
valueStores a named value (variable).
equalsTrue if two values are equal.
greaterTrue if the first value is greater than the second.
lesserTrue if the first value is less than the second.
hasTrue if the object has a specified property/flag/mark.
lacksTrue if the object lacks a specified property/flag/mark.
selfTrue if the object is the caster.
touchedTrue if the inscribed surface is touched.
movedTrue if the object moved.
attackedTrue if the object attacked.
damagedTrue if the object took damage.
healedTrue if the object was healed.
castTrue if the object casts a spell.
chanceRoll 1d10 and return the number (usable with comparisons like greater/lesser/equals).
directionSpecifies a direction (up/down/north/south/east/west). You may also use Xeran direction runes for this.
throw (Type)Specifies the saving throw type used by an effect.
throw (Value)Specifies the save DC. The DC cannot exceed 10 + Vindroth School Level.
delayDelays an effect in 6-second intervals, max 18 × Vindroth School Level.

Vindroth Effects

RuneEffect
PowerDeal 1d4 damage (damage type defined by the attached Xeran element, if any) (counts as a spell attack (unless otherwise specified), roll 1d20 and add Material Modifier for to hit).
Heal (Vitality)Heal 1d4 Vitality.
Heal (Health)Heal 1d4 Hit Points.
CreateCreate a specified Xeran element.
ResistGrant resistance to a specified damage type.
VulnerableGrant vulnerability to a specified damage type.
DispelCounterspell all castings in range with mana cost lower than 20 × Vindroth School Level.
AddAdd a specified number to a specified value (number is defined by mana put into the rune during inscription).
SubtractSubtract a specified number from a specified value (number is defined by mana put into the rune during inscription).
MultiplyMultiply a specified value by a specified number (number is defined by mana put into the rune during inscription).
DivideDivide a specified value by a specified number (number is defined by mana put into the rune during inscription).
MarkApply a named mark to a target. The mark is invisible, but can be detected by has.
PushMove the target in the specified direction (Strength save 10 + Vindroth School Level).
TrapApply the Prone condition (Strength save 10 + Vindroth School Level).

Example Castings

Simple Fire Trap (Entry Trigger)

Structure

If object (any) entered size (5) shape (sphere)
Than Power (Xeran Fire)

What it Does
Deals 1d4 Fire damage to any creature that enters a 5 ft sphere around the inscription.

Structure Use

  • If/Than blocks: 1
  • Logic connectors: 0

Minimum Vindroth School Level

  • 1

Notes
Baseline Vindroth usage. This is the canonical “rune trap” example.

Healing Plate (Touch Trigger)

Structure

If object (self) touched
Than Heal

What it Does
When the caster touches the inscribed surface, they heal 1d4 HP.

Structure Use

  • If/Than blocks: 1
  • Logic connectors: 0

Minimum Vindroth School Level

  • 1

Directional Force Push Trap

Structure

If object (any) entered size (5) shape (sphere)
Than Push direction (north)

What it Does
Any creature entering the area must make a Strength save or be pushed north.

Structure Use

  • If/Than blocks: 1
  • Logic connectors: 0

Minimum Vindroth School Level

  • 1

Conditional Trap with Else Clause

Structure

If object (self) entered size (5) shape (sphere)
Than Heal (Vitality)
Else Power (Xeran Necrotic)

What it Does

  • If the caster enters: heal 1d4 Vitality
  • Otherwise: intruders take 1d4 Necrotic damage

Structure Use

  • If/Than blocks: 1
  • Else clauses: 1

Minimum Vindroth School Level

  • 2

Mana-Scaling Kill Zone

Structure

While object (any) within size (10) shape (sphere)
Than Power (Xeran Fire)
And Power (Xeran Fire)
And Power (Xeran Fire)

What it Does
Deals 3d4 Fire damage to all creatures within the area each turn until they leave area

Structure Use

  • If/Than blocks: 1 (while)
  • Logic connectors: 2 (extra Power via And)

Minimum Vindroth School Level

  • 3

Damage with Save and Delay

Structure

If object (any) entered size (5) shape (sphere)
Than delay (6)
And Power (Xeran Fire)
And throw (Dex)
And throw (10)

What it Does
After 6 seconds, deals 1d4 Fire damage to creatures in the area, Dex save (10) to resist.

Structure Use

  • If/Than blocks: 1
  • Logic connectors: 3 (delay, And Power, And throw)

Minimum Vindroth School Level

  • 4

Mark-and-Execute Rune

Structure

If object (any) entered size (5) shape (sphere)
Than Mark (prey)

If object (any) has (prey)
And attacked
Than Power (Xeran Fire)
And Power (Xeran Fire)

What it Does

  • First trigger: entering marks the target.
  • Second trigger: if a marked target attacks, they take 2d4 Fire damage.

Structure Use

  • If/Than blocks: 2
  • Logic connectors: 2

Minimum Vindroth School Level

  • 4

Smart Defense Engine

Structure

If object (any) entered size (20) shape (sphere)
And lacks (ally)
Than Mark (hostile)

If object (any) has (hostile)
And cast
Than Dispel

If object (any) has (hostile)
And attacked
Than Power (Xeran Force)
And Power (Xeran Force)
And throw (Str)
And throw (15)

Else Heal

What it Does

  • Marks non-allies entering the area
  • Dispels hostile spellcasting
  • Punishes hostile attacks with 2d4 Force damage (Strength save)
  • Heals allies when no hostile condition is met

Structure Use

  • If/Than blocks: 3
  • And connectors: 6
  • Else: 1

Minimum Vindroth School Level

  • 10
